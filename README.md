# 🍻 Beviti Questo!

Gioco alcolico competitivo per due persone. 35 carte, 4 categorie.

## Deploy su Netlify (opzione A — drag & drop, più semplice)

```bash
npm install
npm run build
```

Vai su [netlify.com/drop](https://app.netlify.com/drop) e trascina la cartella `dist/`.

## Deploy su Netlify (opzione B — da GitHub)

1. Carica questa cartella su un repo GitHub
2. Vai su [netlify.com](https://netlify.com) → "Add new site" → "Import from Git"
3. Seleziona il repo — Netlify legge automaticamente il `netlify.toml` e fa tutto da solo

## Sviluppo locale

```bash
npm install
npm run dev
```

Apri [http://localhost:5173](http://localhost:5173)
